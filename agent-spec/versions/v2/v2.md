# Sema4.ai Agent Package Specification v2

## Agent Package File Structure

An Agent Bundle is a directory structure that contains the following components:

```
my-agent[.zip]
├── actions
│   └── MyActions
│       └── geo-actions # folder with sources
│       └── cool-shared-action
│           └── <action package files>
│   └── Sema4.ai
│       └── search
│           └── 1.0.0.zip
├── agent-spec.yaml
├── knowledge
│   └── Knowledge1.pdf
└── runbook.md
```

The formal specification of the `agent-spec.yaml` is available in [agent-package-specification-v2.json](agent-package-specification-v2.json).

An example of an agent spec is available in [example/agent-spec.yaml](example/agent-spec.yaml).
